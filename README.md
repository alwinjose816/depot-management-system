# Smart Depot Management & Inventory Optimization System

An integrated digital solution designed to streamline depot operations, improve inventory visibility, and enhance decision-making through multiple interconnected applications.

---

## Project Overview

This system was developed to manage and optimize depot operations by integrating inventory tracking, employee monitoring, and administrative workflows into a single digital ecosystem. The solution leverages both Python and JavaScript-based applications along with cloud APIs for real-time functionality.

---

## System Modules

### 1. Depot Management App (Python)

Handles inventory tracking, damage reporting, and stock monitoring
🔗 https://github.com/alwinjose816/depot_management_app

---

### 2. Admin App (JavaScript)

Manages approvals, monitoring, and administrative workflows
🔗 https://github.com/alwinjose816/admin-app

---

### 3. Employee Tracker (JavaScript)

Tracks employee location and activity for operational visibility
🔗 https://github.com/alwinjose816/employee-tracker

---

### 4. Saathi Model App (Python)

Supports analytics and intelligent decision-making
🔗 https://github.com/alwinjose816/saathi-model-app

## Technologies Used

### Backend & Processing

* Python (Streamlit, data processing, business logic)

### Frontend & Application Layer

* JavaScript, React (UI development, workflows, interactivity)

### Database & Storage

* Supabase (Database & authentication)
* SQL (Data management)

### Data & Analytics

* Power BI / Excel (Dashboards & reporting)

### Development Tools

* Visual Studio Code
* GitHub (Version control)

### Deployment

* Vercel (Frontend deployment)
* Streamlit (App hosting)

---

## APIs & Integrations

### 🔹 Google Cloud APIs

* Geocoding API (Location conversion & tracking)
* Maps JavaScript API (Map visualization)
* Places API (Location search & details)
* Routes API (Navigation & route optimization)
* Maps Embed API (Map embedding)

### 🔹 Cloud Services

* BigQuery API (Data analysis)
* Cloud Storage API (File storage)
* Cloud SQL (Database services)
* Cloud Logging API (System monitoring)
* Cloud Monitoring API (Performance tracking)

### 🔹 External APIs

* Weather API (Real-time weather data integration)

---

## System Architecture

This system follows a modular architecture:

* Python-based applications handle backend processing and analytics
* JavaScript-based applications manage user interface and workflows
* APIs enable real-time data exchange and location services
* Cloud services ensure scalability, monitoring, and data storage

---

## Key Features

* Inventory tracking and optimization
* Damage reporting with approval workflow
* Employee tracking and monitoring
* Admin dashboard for control and decision-making
* Real-time location tracking using Maps APIs
* Data-driven insights and reporting

---

## Business Impact

* Improved visibility of depot operations

* Faster decision-making through real-time data

To build a scalable and efficient digital system that integrates multiple operational functions into a unified platform, improving supply chain efficiency and depot management performance.
